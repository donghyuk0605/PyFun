## 📚 교보문고 데이터 분석 프로젝트
**파일명:** `koNLPy_wordcloud_kyobo.ipynb`  
**기술 스택:** `KoNLPy`, `WordCloud`, `Pandas`, `Matplotlib`, `Seaborn`

### 🔍 프로젝트 개요
이 프로젝트에서는 **교보문고의 상품 데이터**를 활용하여 주요 키워드를 추출하고, 이를 다양한 시각화 기법으로 분석합니다.  
`KoNLPy`를 활용한 한국어 형태소 분석과 `WordCloud`로 워드클라우드 시각화를 구현하여 데이터를 효과적으로 해석합니다.

### 🛠️ 주요 기능
1. **데이터 전처리:**  
   - 불필요한 기호 제거, 숫자형 데이터 변환 등.  
2. **키워드 추출:**  
   - KoNLPy의 `Okt`를 사용해 상품명에서 주요 명사를 추출.  
3. **시각화:**  
   - `Seaborn`으로 상위 키워드 빈도 그래프 생성.  
   - `WordCloud`로 인기 키워드를 직관적으로 표현.

## 🤖 텐서플로로 블랙기업 사장 AI 프로젝트
**파일명:** `tensorflow_black_company_ai.ipynb`  
**기술 스택:** `TensorFlow`, `Keras`, `Natural Language Processing (NLP)`

### 🔍 프로젝트 개요
이 프로젝트에서는 텐서플로우(TensorFlow)를 사용해 **블랙기업 사장의 발언**에 따른 직원 반응(퇴사 vs 충성)을 예측하는 AI 모델을 구축합니다.  
LSTM(Long Short-Term Memory)을 활용하여 텍스트 데이터를 학습하며, 감정 분석과 자연어 처리를 적용합니다.

### 🛠️ 주요 기능
1. **데이터 준비:**  
   - 블랙기업 사장의 발언 데이터를 "퇴사 발언"과 "충성 발언"으로 분류.  
2. **텍스트 전처리:**  
   - `Tokenizer`를 사용해 텍스트를 숫자로 변환하고 패딩 처리.  
3. **모델 학습:**  
   - `Embedding`과 `LSTM` 레이어를 포함한 텍스트 분류 모델 구성.  
4. **결과 예측:**  
   - 새로운 문장을 입력하면 "퇴사" 또는 "충성"을 예측.
